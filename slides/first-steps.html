First Steps

d3.select("#d3Sample")
.append("p")
.text("Hello World");
  • Access the d3 object
  • ... call the select function to create a selection
  • ... pass in a CSS selector to specify what elements to put in this Selection
  • ... for all elements in the selector append an element
  • ... in this case a paragraph element
  • ... this returns a new Selection, on that Selection set the text
  • ... to Hello World!
